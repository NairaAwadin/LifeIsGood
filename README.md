# LifeIsGood

- Assistant Intelligent de recherche de logement

## Informations importantes : 
 
 L'application ne va pas fonctionner sans base de données et models

## Installer Streamlit et les dépendances

py -m venv venv <br>
source venv/scripts/activate <br>
pip install requirements <br>

## Run le projet

streamlit run app.py
